## Features
- Arcaea
  - Best30 query
  - Song information query
  - Chart preview
  - Potential calculate
  - ...still working!
- C# code runner
- Random cat/capoo image
- Wolfram Alpha query
- Translate

...etc.

Documentation can be found [here](https://docs.blacksoil.top/YukiChan/) (Simplified Chinese only).
